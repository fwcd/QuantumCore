# QuantumCore
A quantum computing library for Java.

[![Build Status](https://travis-ci.org/fwcd/QuantumCore.svg?branch=master)](https://travis-ci.org/fwcd/QuantumCore)

## Features
* Quantum simulator
* Entangled Qubit registers
* Measurement
* Quantum Gates
	* Hadamard
	* Pauli X, Y and Z
	* SqrtNOT
	* CNOT
	* SqrtSwap
	* Swap
	* CCNOT (Toffoli gate)
	* CSwap (Fredkin gate)
